# Power BI Sales Dashboard

## Overview
This project is an **interactive Sales Dashboard** built in **Power BI** to analyze sales and profit across different categories, regions, and customer segments.

## Features
- Cleaned and transformed data using **Power Query**
- Created **data model** and applied **DAX** for calculations
- Visualized **sales, profit, and order trends** using bar charts, line charts, and KPIs
- Added **filters, slicers, and drill-downs** for interactive analysis

## Tools Used
Power BI | DAX | Power Query | Excel | Data Visualization
